1) Realizar as seguintes operações no Git
 Criar um repositório: git init 
 Fazer alterações em um arquivo e gerar versões nele com commit: git add . || git commit -m "atualizações"
 Recuperar uma versão a partir de um commit específico: git reset --hard "5250dda7002af9e8e309242a6d59482920d23591"
 Clonar um repositório local em outra pasta: ?
 Clonar um repositório remoto: git clone git@github.com:matheustimbo/vivENDO-App.git 
 Subir uma versão para o Github: git push -u origin
